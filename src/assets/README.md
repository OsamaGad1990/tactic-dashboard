# Assets Directory

This directory contains brand assets for Tactic Portal.

## Structure
```
assets/
├── logo/
│   ├── logo-dark.svg      # Logo for dark mode (Gold/Light colored)
│   ├── logo-light.svg     # Logo for light mode (Dark/Blue colored)
│   └── logo-icon.svg      # Icon-only version
├── backgrounds/
│   ├── login-bg-dark.svg  # Login background for dark mode
│   └── login-bg-light.svg # Login background for light mode
└── README.md
```

## Usage
- Place your logo files in the `logo/` directory
- Place background images in the `backgrounds/` directory
- Supported formats: SVG (recommended), PNG, WebP

## Naming Convention
- Use `-dark` suffix for dark mode variants
- Use `-light` suffix for light mode variants
